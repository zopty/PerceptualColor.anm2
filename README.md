# PerceptualColor.anm2
AviUtl2に知覚的な混色とグラデーションを追加するスクリプト

beta27にて動作確認済

# 導入手順

1. [リリースページ](https://github.com/zopty/PerceptualColor.anm2/releases/latest/) から `PerceptualColor.anm2` をダウンロード
2. AviUtl2 の画面に `PerceptualColor.anm2` をドラッグ＆ドロップ
3. `このプラグイン・スクリプトを信頼して使用する` をクリック
4. `色調整` に追加されます
